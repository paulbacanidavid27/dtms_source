Retention Disposable Schedules

This is a process that will determine the life expentancy of a record. It consist of 2 step process. The active period and storage period. These periods will be discussed further below.

How to enable Retention Process

1. Every record retention period will depend on the document type. To enable RDS process, go to Admin -> Document Types screen. Enable the RDS by clicking on Enable Retention check box.
2. Setup the Active Period and Storage Period. Active period can be in Days, Months or Years. Set this based on requirement of the document type.
3. Storage Period will be computed based on the this value. It can be in Days, Months or Years.

How Retention Disposable Schedule Process Works

1. When a document has been archived and the Document Index field called Date Received by Agency has been filled with date then the active period and storage period will be computed.

Computation of Periods

Active Period

Start of Active Period = Date Received By Agenycy
End of Active Period = Date Received By Agenycy + Active Period Based on the Document Type
Start of Storage Period = End of Active Period + 1 day
End of Storage Period = Start of Storage Period + Storage Period Based on the Document Type

Notes

Archiving of Documents can be done in two ways
1. Upon uploading of documents.
2. In the View screen where you can edit the properties of a document specifically the last action to set it to Archived.
3. When you set the document to Archived and the Date Received by Agency has not been setup the Active and Storage period will not be setup.
Date Received by Agency field
1. Go to the View screen and click on Document Index tab. Look for the "Date Received By Agency" field and fill this with date. Once you click on save and the document status has been set to Archived. the retention period will be triggered.

Retention Period Storage Due Date

1. Once a document has reach the end of Storage Period, a background process will execute to move the file (pdf) of the record to another folder and the Document status will be set to purge.
2. Purged documents can still be search but cannot be viewed anymore

Email Notification

Email notification for all the documents that will be purged 5 days before will be sent to the Purge Point Person.

Where to setup email recepients

Go to Admin -> Settings

How To Cancel the Purging of Documents

You can set the document status to "Open" or you can edit the Date Received by Agency index field.
